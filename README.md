# Bioinfo Study Portfolio

Python을 활용한 생물정보학(Bioinformatics) 기초 분석 및 데이터 처리 학습 기록입니다.

---

## Projects 

### 1. DNA GC Content Analyzer
- DNA sequence의 길이와 GC content 계산
- 반복문과 조건문을 활용한 염기 빈도 분석
- 결과를 txt 파일로 저장

### 2. Interactive DNA Analysis Tool
- 사용자 입력 기반 DNA sequence 분석 프로그램
- 소문자 입력 자동 변환 및 비정상 염기 검증 기능 구현
- A/T/G/C 외 문자 검증
- GC content 비율 기반 sequence 특성 분류
- 분석 결과를 log 파일로 저장
